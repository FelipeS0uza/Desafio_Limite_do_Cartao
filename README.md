#Desafio Limite do Cartão com JAVA

Participei de um desafio que consiste em instaciar um cartão de crédito com um limite e fazer compras, o algoritmo dará a opção de continuar ou sair e irá adicionando as compras em uma lista que será mostrada no final, além de sair sozinho quando a próxima compra ultrapassa o limite do cartão.
O algoritmo em si é simples, mas como estou iniciando na linguagem Java, foi ótimo para testar meus conhecimentos e até mesmo aprender melhor sobre a interface da IDE IntelliJ.
